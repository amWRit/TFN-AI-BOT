## New Usage

### From Project Root:

```bash
# Install Python dependencies
pip install -r scripts/requirements.txt

# Preprocess PDFs
python scripts/preprocess_docs.py

# Install Node dependencies
npm install

# Run dev server
npm run dev
```

## From Scripts Folder:

```bash
# Install Python dependencies
cd scripts
pip install -r requirements.txt

# Preprocess PDFs
python preprocess_docs.py

# This will:
# 1. Load PDFs from ../data/
# 2. Process and chunk them
# 3. Save to ../public/tfn-documents.json
```
