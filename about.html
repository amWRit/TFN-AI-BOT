A cutting-edge chatbot that combines structured and unstructured data retrieval with AI-powered generation for Teach For Nepal

  • Multi-Source Data: Processes PDFs, staff records, alumni data, schools, and partner information
  • AWS Bedrock Integration: Leverages Nova Lite LLM and Titan v2 embeddings
  • Modern Tech Stack: Next.js 14+, Python preprocessing, FAISS vector store
  • Intelligent Pagination: "View More" functionality for large result sets

Data Flow Pipeline

1. Preprocessing (Offline)

  • Run python scripts/pipeline.py to orchestrate full pipeline
  • Scraper collects structured data from web sources
  • PDF extraction and chunking for unstructured documents
  • FAISS index building with Bedrock embeddings
  • Output: vector store and JSON files in public/

2. Runtime (Online)

  • User submits query through web interface
  • Backend loads FAISS index and structured data
  • Hybrid search determines query type and retrieves context
  • LLM generates answer with source citations
  • Frontend displays adaptive cards with pagination
